# Muhammad Hassan - Software Engineer Portfolio

A modern, high-performance portfolio website built with **Angular 19** and **Tailwind CSS v4**. Designed with a strict **Dark Mode** aesthetic, featuring smooth animations, responsive layout, and a clean code architecture.

## Tech Stack

- **Framework:** [Angular 19](https://angular.io/) (Latest features: Signals, Control Flow, Standalone Components)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) (Oxide engine, CSS variables)
- **Animations:** AOS (Animate On Scroll) & CSS Native Transitions
- **Deployment:** Netlify Ready
- **Icons:** FontAwesome / Custom SVG

## Features

- **Dark Mode:** A premium, consistent dark theme across all components.
- **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop.
- **High Performance:** Lazy loading, optimized assets, and modern build tooling.
- **Modern UI:**
  - Glassmorphism effects (backdrop-blur).
  - Gradient borders and text.
  - Interactive hover states.
- **Sections:**
  - **Hero:** Dynamic introduction.
  - **Skills:** Categorized technical expertise.
  - **Projects:** detailed case studies of work.
  - **Experience:** Professional journey timeline.
  - **Contact:** Functional form layout.

## Getting Started

### Prerequisites

- **Node.js** (v18 or higher recommended)
- **npm** or **yarn**
- **Angular CLI** (`npm install -g @angular/cli`)

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Muhammad-Hassan-522/Portfolio.git
    cd Portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    ng serve
    ```
    Navigate to `http://localhost:4200/`.
